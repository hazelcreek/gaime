# Style Preset Ideas

Future style presets to consider creating. Each includes a brief description of the visual aesthetic.

---

## Historical Art Movements

| Style | Description |
|-------|-------------|
| **Ukiyo-e** | Japanese woodblock prints—flat colors, bold outlines, wave patterns, Hokusai/Hiroshige aesthetic |
| **Art Nouveau** | Alphonse Mucha—flowing organic lines, decorative borders, elegant female figures, floral motifs |
| **Art Deco** | 1920s geometric glamour, gold and black, sunburst patterns, Gatsby-era luxury |
| **Soviet Constructivism** | Bold propaganda posters, red/black/cream, diagonal compositions, heroic workers |
| **Bauhaus** | Primary colors, geometric shapes, clean modernist design, functional beauty |
| **Romanticism** | Caspar David Friedrich—sublime landscapes, tiny figures against vast nature, dramatic skies |
| **Pre-Raphaelite** | Lush medieval romanticism, rich jewel tones, flowing hair, Rossetti/Waterhouse |
| **Impressionism** | Monet/Renoir—dappled light, visible brushstrokes, outdoor scenes, fleeting moments |
| **Surrealism** | Dalí/Magritte—dreamscapes, impossible juxtapositions, melting reality |
| **Expressionism** | Munch/Kirchner—emotional distortion, bold colors, psychological intensity |

## Notes

- Prioritize styles that translate well to first-person adventure game scenes
- Consider how NPCs will integrate with each style
- Some styles (like Obra Dinn's 1-bit) may need special handling
- Test new presets across multiple scene types before finalizing
